Плановые формы

  • 1).Информация о проекте
  • 2).Информация о заказчике и исполнителе проекта
  • 3).Плановые сроки завершения этапов(стадий) прохождеия инновационного проекта
  • 4).Плановые объемы и источники финансирования инновационного проекта
  • 5).Плановые объемы финансирования проекта за счет средств инностранных инвесторов по странам происхождения инвестиций
  • 6).Плановые общие инвестиционные затраты по инновационному проекту
  • 7).Наименование продукции, создаваемой в результате реализации проекта, и планы по выпуску
  • 8).Плановый объем продукции, созданной в результате реализации инновационного проекта
  • 9).Плановое количество модернизируемых и вновь создаваемых рабочих мест
  • 10).Плановые показатели эффективности проекта по годам

Квартальные формы

  • 1).Этапы(стадии) прохождения инновационных проектов
  • 2).Объемы и источники финансирования инновационных проектов
  • 3).Инвестиционные затраты по инновационным проектам
  • 4).Приобретение и использование объектов промышленной собственности
  • 5).Показатели производства продукции, созданной в результате реализации инновационных проектов
  • 6).Создание ( модернизация) рабочих мест в результате инновационных проектов
  • 7).Показатели эффективности проектов за отчетный период(заполняется по итогам года)
  • 8).Инвестиционная импортоемкость проекта
  • 9).Описание хода реализации проектов
